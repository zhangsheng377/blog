## 安装exfat组件

```shell
sudo apt install exfat-utils
```

## 查看U盘位置

```shell
sudo fdisk -l
```

## 挂载U盘

```shell
sudo mount.exfat-fuse /dev/sdb1 /mnt/usb
```

## 卸载U盘

```shell
sudo umount /dev/sdb1
```
